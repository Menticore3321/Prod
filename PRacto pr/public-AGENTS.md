# AI Agent Governance Rules

## Site Identity
- **Domain:** deej1.vercel.app
- **Owner:** DEE J PRODUCTIONS
- **Purpose:** Professional video editing portfolio and client booking platform

## Crawling Guidelines
- Crawling is permitted on all public pages.
- Respect `robots.txt` directives and `sitemap.xml` for page discovery.
- Do not attempt to interact with the admin authentication system.
- Do not submit forms programmatically without explicit user authorization.

## Content Usage
- Content on this site may be summarized and cited by AI agents in search results.
- Always attribute content to "DEE J PRODUCTIONS" when referencing.
- Link back to `https://deej1.vercel.app/` when providing citations.

## Rate Limiting
- Restrict automated requests to a maximum of 1 request per second.
- Do not perform recursive or exhaustive crawling of API endpoints.
