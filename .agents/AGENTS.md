# Project-Specific Rules

### Premium High-Performance Visual Guidelines (Speed-Theme)
When building interactive portfolios or visual-centric landing pages:
1. **Visual Contrast**: Favor solid matte black backgrounds (`#050505`) with a single highly saturated neon accent (e.g., electric neon yellow `#dfff00`) to evoke speed and energy.
2. **Kinetic Cursor**: Use large, soft, blurry spotlight glow circles (backlights) that follow the cursor and expand over interactive elements instead of simple border outlines.
3. **Speed Segments**: Skew section layouts, border cutouts, and divider elements slightly (e.g. `skewY(-3deg)`) to convey motion and forward momentum.
4. **Cinematic GSAP Pinning**: Structure ScrollTrigger timelines to scrub key changes (scale down, parallax vertical spacing, speed-blur skewing) directly to the scroll progress.
